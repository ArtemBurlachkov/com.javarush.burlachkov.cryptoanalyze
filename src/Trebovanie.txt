Техническое задание
Разработать программу, позволяющую шифровать и расшифровывать текст с использованием шифра Цезаря.

Программа должна поддерживать несколько режимов работы
1) Шифровка текста с помощью ключа
2) Расшифровка текста с помощью ключа
3) Расшифровка текста с помощью brute force (перебор всех вариантов)

- программа должна укзанный пользователем файл
- проделовать с ним одно и тоже
- после этого создавать новый файл с результатом
- обеспечивать обработку больших файлов - Читаем через буфер
- иметь валидацию входных файлов
